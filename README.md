Manypoint
=========

Emacs mode that lets you move around large files by setting multiple points.

Example:
![](https://github.com/andreasjansson/manypoint/blob/master/github-assets/example.gif)

Key bindings:
 * M-SPC s
   - save current point
 * M-SPC j
   - jump to a point
 * M-SPC c
   - clear all points
